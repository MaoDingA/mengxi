# Changelog

All notable changes to this project will be documented in this file.

## [5.0.0.0] - 2026-03-31

### Added

- **AI 代理框架**: 新增 `mengxi-agent` crate，实现完整的 LLM Provider 抽象层，支持 Claude 和 OpenAI 兼容 API 的 SSE 流式响应
- **工具执行循环**: 代理编排层实现工具调用循环（Story 1.4），支持多轮工具调用与上下文管理
- **TUI 聊天界面**: 集成 Agent 框架到 CLI，实现交互式 TUI 分屏布局（对话面板 + 结果面板），支持 Markdown 渲染和命令面板
- **22 个代理工具**: 完成搜索分析（search, analyze, export）、项目管理（project, fingerprint, tag）、系统工具等核心工具的功能实现
- **4 个子代理**: 注册对比代理（compare）、审核代理（review）等子代理工具，支持任务分解与并发管理（subagent crate）
- **会话持久化**: 添加会话持久化功能，基于 SQLite 存储对话历史，支持分支管理和消息压缩
- **LUT 智能编辑**: 新增 LUT 编辑及差异工具，支持区域哈希校验和撤销操作
- **系统提示词**: 内置调色科学领域知识上下文和工具使用指南，增强代理专业能力
- **智能对话代理**: 完成智能对话代理及对应架构支持，整合所有组件

### Changed

- 更新依赖项以添加多个新包及版本调整（Ratatui、Crossterm 等 TUI 依赖）

### Fixed

- 修复工具调用 ID 不匹配和消息格式问题（agent tool call ID mismatch）
- 修复代理框架中的关键消息格式错误

## [4.0.0.0] - 2026-03-29

### Added

- **瓦片特征提取**: 实现基于网格的瓦片级特征提取，支持部分参考搜索（partial reference search）
- **瓦片数据层**: 新增 `fingerprint_tiles` 数据表及瓦片 CRUD 操作，集成到导入和重提取管线
- **瓦片级搜索**: 支持瓦片级相似度搜索，提供 spatial（空间模式）和 any（任意匹配）两种搜索模式
- **SLIC 超像素分割**: 纯 Rust 实现的 SLIC 超像素分割算法，用于区域感知搜索
- **区域感知搜索**: 基于 SLIC 超像素的段级搜索，提升局部区域匹配精度
- **空间金字塔匹配**: 实现多分辨率空间金字塔匹配算法，增强搜索的空间感知能力
- **搜索管线集成**: 将空间金字塔匹配集成到搜索管线，实现多层级特征对比

## [3.0.0.0] - 2026-03-29

### Added

- **算法参数化**: 支持可配置的分级特征直方图箱数，统一特征提取管线调用接口
- **特征词汇表**: 提取共享特征管线并配置特征词汇表（vocabulary），标准化特征表示
- **HNSW 向量索引**: 实现 HNSW 近似最近邻向量索引，大幅提升高维特征搜索性能
- **批量嵌入生成**: 新增 `embed` 子命令，支持批量嵌入向量生成
- **批量重提取**: 事务包裹的批量特征重提取，提升性能
- **特征对比**: 新增 `compare` 子命令，支持并排指纹特征分析
- **跨项目一致性**: 新增 `consistency` 子命令，支持跨项目的色彩一致性分析
- **数据集验证**: 新增 `validate-dataset` 子命令，支持评估数据集格式合规检查
- **JSON 输出标准化**: 标准化所有命令的 JSON 输出格式，统一 `--format` 标志

### Changed

- 扩展色彩矩特征数组至 12 维，增强色彩特征表达能力
- MoonBit 侧直方图边缘使用线性插值软钳位（soft clamp），提升特征鲁棒性

### Fixed

- 修复 12 维色彩矩的迁移测试
- 修复 JSON stdout/stderr 分离问题

## [2.0.0.0] - 2026-03-27

### Added

- **Oklab 色彩科学**: 新增 sRGB/Oklab 双向转换、Linear sRGB/Oklab FFI 接口、RGB 批量转 Oklab，完整的感知均匀色彩空间支持
- **ACEScct 色彩空间**: 新增 ACEScct 和 Oklab 颜色空间转换功能
- **颜色空间验证**: 新增 `mengxi validate` 命令及 `--full` 数值安全测试，验证色彩空间转换精度
- **调色特征提取**: 新增 Oklab 直方图和色彩矩 FFI 提取功能，通过 MoonBit 桥接实现高性能计算
- **BLOB 序列化**: 新增调色特征的 BLOB 序列化/反序列化模块，优化数据库存储
- **降采样模块**: 新增 `downsample` 模块，支持 RGB 图像降采样
- **Bhattacharyya 距离**: 实现 Bhattacharyya 距离候选排序（Story 3.1），提升搜索排序质量
- **混合加权搜索**: 实现混合信号加权搜索功能，支持多种特征联合评分
- **混合搜索模式**: 支持混合搜索模式及权重自定义，用户可调节各特征权重
- **色彩空间差异检测**: 添加色彩空间差异检测并在结果中展示匹配警告
- **特征状态管理**: 添加指纹特征过期自动重计算及限流机制，新增 `feature_status` 列（migration 018）
- **评分描述**: 添加人类可读的评分描述字段并支持显示
- **特征重提取**: 实现从源文件重提取调色特征（Story 5.4）
- **项目级配置**: 实现项目级配置级联（project-level config cascade）
- **交互式标签输入**: 添加交互式标签输入功能
- **数据库浏览**: 添加数据库浏览查询子命令及相关功能

### Changed

- 重构 `grading_features` BLOB 拆分为独立列，提升查询灵活性
- 重构 search 模块中的批量标签加载函数
- 添加调色特征相关错误类型，改进 RGB 降采样接口

- 修复 Linear round-trip ΔE 测试恒等式问题
- 修复像素数据长度溢出和索引错误

- 修复调色特征模块代码审查发现的 4 个问题
- 修复调色特征 Story 2.2 代码审查补丁
- 修复 stale 状态指纹特征状态更新错误

## [1.0.0.0] - 2026-03-22

### Added

- **CLI 框架**: 9 个子命令的完整 CLI 工具（import, search, export, info, tag, lut-diff, lut-dep, stats, config），支持交互模式和脚本模式，JSON/文本双格式输出
- **项目导入**: 支持 DPX（8/10/12/16-bit）、EXR（half-float、多压缩格式）、MOV 文件格式自动检测与导入，支持断点续传和进度显示
- **色彩指纹提取**: 通过 MoonBit FFI 桥接，从导入文件中提取直方图、色彩空间分布、关键帧色彩特征等指纹数据
- **ACES 1.3 色彩科学**: 完整的 ACES 色彩空间转换引擎，支持 ACEScg、ACES2065-1、Rec.709 之间的数学精确转换
- **LUT 管理**: 多格式 LUT 文件读写（.cube/.3dl/.look/.csp/ASC-CDL）、PowerGrade 只读解析、LUT 导出、LUT 对比差异报告、LUT 依赖追踪
- **直方图搜索**: 基于色彩直方图的相似度搜索，支持项目范围过滤和结果数量配置
- **图像嵌入搜索**: Python AI 子进程 + ONNX 推理，基于图像参考的语义视觉相似度搜索，嵌入向量缓存
- **标签搜索**: 基于语义标签的搜索、搜索结果接受/拒绝反馈
- **AI 标签生成**: 自动为索引项目生成语义标签，支持事后批量生成
- **手动标签管理**: 标签的增删改查、来源追踪（AI/手动）、标签重命名
- **标签校准学习**: AI 从用户标签修正中学习，改进未来标签生成的个性化词汇
- **会话跟踪**: CLI 使用会话记录，命令序列和时间测量
- **使用统计**: 搜索命中率、校准活动指标、词汇增长率和复用率趋势、用户维度统计与报表
- **嵌入式数据库**: SQLite (rusqlite) WAL 模式，16 个迁移文件，8 张数据表，13 个索引
- **Python AI 子进程**: JSON-RPC 通信，空闲超时自动回收，崩溃自动重启，模型可插拔
- **TOML 配置**: 单文件配置 `~/.mengxi/config`，合理的默认值
